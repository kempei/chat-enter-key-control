# 実装計画

このドキュメントは、Chat Enter Key Control拡張機能をChrome Web Storeで公式リリースするための実装タスクリストです。

## タスクリスト

- [x] 1. プライバシーポリシーの作成
  - `store-assets/privacy-policy/`ディレクトリを作成
  - 日本語版と英語版のプライバシーポリシーを作成
  - Chrome Web Storeの必須項目（データ収集、保存、使用、共有、ユーザーの権利）を含める
  - _要件: 1.1, 1.3, 1.4_

- [x] 2. プライバシーポリシーの公開設定
  - GitHub Pagesまたはリポジトリ内での公開方法を決定
  - プライバシーポリシーのURLを確認
  - HTTPSでアクセス可能であることを確認
  - _要件: 1.2_

- [ ] 3. ストアリスティングコンテンツの作成
  - `store-assets/descriptions/`ディレクトリを作成
  - 日本語版の簡潔な要約を作成（132文字以内）
  - 英語版の簡潔な要約を作成（132文字以内）
  - 日本語版の詳細説明を作成（500-1000文字推奨）
  - 英語版の詳細説明を作成（500-1000文字推奨）
  - 検索キーワードをリストアップ
  - _要件: 2.1, 2.2, 2.3, 2.4_

- [ ] 4. スクリーンショットの作成
  - `store-assets/screenshots/ja/`と`store-assets/screenshots/en/`ディレクトリを作成
  - ポップアップUI全体のスクリーンショットを撮影（日本語版・英語版）
  - 送信キー設定画面のスクリーンショットを撮影（日本語版・英語版）
  - ChatGPTでの動作例のスクリーンショットを撮影（日本語版・英語版）
  - URLパターンの追加/削除操作のスクリーンショットを撮影（日本語版・英語版）
  - 必要に応じて注釈を追加
  - 1280x800ピクセルにリサイズ
  - PNG形式で保存
  - _要件: 3.1, 3.2, 3.3, 3.6_

- [ ] 5. プロモーション画像の作成

  - 小タイル画像（440x280ピクセル）を作成（日本語版・英語版）
  - 拡張機能のロゴと主要機能の説明を含める
  - 視覚的に魅力的なデザインにする
  - PNG形式で保存
  - _要件: 3.4, 3.6_

- [ ]* 5.1 大タイル画像の作成（オプション）
  - 大タイル画像（1400x560ピクセル）を作成（日本語版・英語版）
  - より詳細な機能説明やビジュアルを含める
  - PNG形式で保存
  - _要件: 3.5_

- [ ] 6. manifest.jsonの確認と最適化
  - 必須フィールド（name、version、description、icons、manifest_version）が含まれていることを確認
  - descriptionが132文字以内であることを確認
  - homepage_urlを追加（GitHubリポジトリのURL）

  - _要件: 4.1, 4.2, 4.3, 4.4, 4.5_

- [ ] 7. リリースパッケージ作成スクリプトの作成

  - ZIPファイルを作成するロジックを実装

## 注意事項

- タスク10、11、12はテスト関連のタスクで、オプションとしてマークされています。コア機能の実装を優先する場合はスキップできます。
- タスク5.1（大タイル画像の作成）はオプションです。小タイル画像のみでも審査は通過できます。
- 審査には数日〜数週間かかる場合があります。審査中に追加情報を求められる場合があるため、メールを確認してください。
- 審査が却下された場合は、理由を確認して修正し、再度申請してください。

## 次のステップ

このタスクリストに従って、Chrome Web Storeリリースの準備を進めてください。各タスクを完了したら、チェックボックスにチェックを入れてください。

すべてのタスクが完了したら、審査申請を行い、審査結果を待ちます。審査が通過したら、拡張機能が公開され、ユーザーがChrome Web Storeからインストールできるようになります。
